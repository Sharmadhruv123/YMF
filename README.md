# Youth Motivation Forum (YMF)

Welcome to the **Youth Motivation Forum (YMF)** landing page! 

YMF is India's largest youth motivation movement, dedicated to empowering youth to realize their true potential and shape the future of society. This project is a modern, responsive landing page showcasing the initiatives, impact, and global network of the Youth Motivation Forum, led by visionary Sajan Shah.

## Features
- **Modern Neo-Brutalism Design**: A clean, vibrant, and highly engaging illustration-based UI.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.
- **Dynamic Animations**: Smooth scroll animations, hover effects, and interactive elements.
- **Global Presence Highlights**: Showcases the worldwide network of YMF support offices.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **Animation**: Custom CSS keyframes and transitions

## Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Sharmadhruv123/YMF.git
   ```
2. Navigate to the project directory:
   ```bash
   cd YMF
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
